
#NBA SALARY INSIGHT

Project Description: We will investigate whether certain factors contributed to a NBA players salary earnings. 

Research Question to Answer: 
  - What attributes contribute to the salaries of NBA players?
    - Salary Cap
    - Player performance
    - Position
    - Age
    - Social Media
